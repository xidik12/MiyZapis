üìã Test Instructions

  1. Setup Coinbase Commerce:
    • Go to Coinbase Commerce
    • Create a test account or use sandbox mode

    • Add them to your .env file as:

  3. Webhook Testing:
    • Use ngrok to expose your local server: ngrok http 3030
    • Set webhook URL in Coinbase Commerce to: https://your-ngrok-url.ngrok.io/api/v1/payments/coinbase-webhook
